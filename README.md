# Railpack

Use webpack/esbuild/bun/rollup to manage JavaScript without the "asset pipeline" in your Rails app.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'railpack'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install railpack
